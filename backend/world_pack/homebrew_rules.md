# Homebrew System Rules & Resolution Logic

## 1. Actions & Skill Checks
- **The Rule of Capability:** If an action aligns perfectly with a character's role (e.g., Kael picking a lock, Jax breaking an iron pipe), the action succeeds automatically.
- **The Cost of Uncertainty:** If an action is risky or outside a character's core skillset, roll a d20 check with the appropriate difficulty.

## 2. Difficulty Tiers
- Easy (DC 8): Routine actions in a character's specialty.
- Moderate (DC 12): Risky actions requiring focus.
- Hard (DC 14): Dangerous actions under pressure.
- Severe (DC 18): Nearly impossible without help.

## 3. Combat Rules
- Enemies have Stamina scores (5-15 for basic foes, 15-25 for elites).
- A successful attack reduces Stamina by 1d6 + relevant modifier.
- At 0 Stamina, the enemy is neutralized.
- Area effects and traps can hit multiple targets.

## 4. Rest & Recovery
- **Short Rest (10 minutes):** Restore 1d4 health, requires safe location.
- **Long Rest (few hours):** Full heal, requires shelter and warmth.
